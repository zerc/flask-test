Flask test job
=====================

Just a test.

### Install
    sudo apt-get install python-pip, sqlite3
    sudo pip install virtualenv

    git clone git@github.com:zerc/flask-test.git flask-test
    cd flask-test
    virtualenv venv
    . venv/bin/activate
    pip install -r pip.req

Run:

    ./run.py

And go to *http://localhost:8000/*
